CV de mon site Web

Mon site web reprend l'idée d'un wiki axée sur la série Netflix "Scott Pilgrim Prend Son Envol" que j'apprécie énormément.
Il contient plusieurs pages :
- La page d'acceuil
- La page "Histoire" qui contient un résumé de la série
- La page "Personnages" qui contient une liste des personnages principaux de la série
- La page "Admin" qui est celle-ci
La page "Admin" est une page qui n'est pas accessible par les utilisateurs normaux, elle est réservée aux administrateurs du site.
